Choose the type of your Marko.
Website is for url links, links to social profiles and so much more. Email is for email addresses. Phone, Viber and WhatsApp make more sense on mobile. Viber and WhatsApp need the phonenumber prefixed with the international code. Window creates a local frame which you can use to surf a website of your choice. Adding a Youtube link in window lets you watch a video without leaving Marko. Using Note you can take a local note, which never leaves your browser. Same applies to post. Selecting post you can create a local website post. Location accepts geo: links or links to Google maps or openstreetmaps. Payment accepts paypal.me and bitcoin: links. Choose pdf to add a pdf to read inside Marko. Download creates a link to a download object. Videocall creates a link you can use to share with the friend(s) you want to get in touch with.
If you are unsure what to try first, choose website, which is the default anyway...

This option lets you choose the type that will be shown for your Marko. Website icon (where applicable) will automatically try to fetch from the internet the favicon (the default site icon) for you to use as Marko icon. Let me choose ..lets you choose a suitable Google material icon or FontAwesome icon for your Marko. This option will also let you define a custom color for the ring around the icon. A random-colored ring will be created otherwise. The option Default will select a predefined suitable material-style icon for your Marko based on its type, which you chose in a previous step.

Add a preferred name (max 11 characters) so that a custom shortlink can be created using spoo.me. If you check the use-twice option your link will work only twice.. Click cancel to create a looong link with parameters without using external services. You can also share this link with friends and family and them clicking on it will add your Marko on their Marko app.
